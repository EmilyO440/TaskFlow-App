# TaskFlow 📝

A modern task management app for organizing tasks, tracking progress, and staying on top of priorities.

---

## ⚡ Features

- Add, edit, and delete tasks  
- Set priority levels (Low, Medium, High)  
- Assign due dates  
- Filter tasks (All / Active / Completed)  
- Track completion progress
